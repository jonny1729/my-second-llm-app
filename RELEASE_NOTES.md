## 📋 システム要件

### 対応環境
- **OS**: Windows 10/11
- **RAM**: 4GB以上推奨
- **ストレージ**: 500MB以上の空き容量
- **ネットワーク**: LLM機能使用時のみ必要

### 技術スペック
- **フレームワーク**: Electron + React + TypeScript
- **データベース**: SQLite
- **アニメーション**: Framer Motion
- **状態管理**: Zustand
